首先，请用户登录控制台完成产品开通。

**使用流程：**

![](../../../../image/Edge-Network-Tunnel/Quick-Start.png)

###### 步骤一：下载SDK

根据技术选型，选择对应的SDK(C、Go)进行下载；

###### 步骤二：终端注册/连接使用

调用SDK-API对终端进行注册，之后可以和其他终端进行连接、发送信息、获取信息等操作，详情请查阅'*SDK文档*';

###### 步骤三：使用详情查看

在产品控制台，方便的查看使用的流量、产生的隧道通路、注册成功的终端设备，并可对其进行基础运维工作。
