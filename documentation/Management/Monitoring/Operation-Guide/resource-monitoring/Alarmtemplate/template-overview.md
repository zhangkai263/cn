# 报警模版

报警模板能够帮助用户快速创建和更新报警规则。报警模板分为默认报警模板和自定义报警模板。

- 默认报警模板是京东云平台根据不同云产品的特性提供的默认模板，方便初次使用云产品的用户快速创建报警规则。
- 自定义报警模板是用户根据自身业务需求的不同创建的报警模板。



## 使用限制

| 限制项                           | 限制规则 |
| -------------------------------- | -------- |
| 自定义模版数量                   | 100个    |
| 单个模版中支持的最大触发条件数量 | 10条     |
