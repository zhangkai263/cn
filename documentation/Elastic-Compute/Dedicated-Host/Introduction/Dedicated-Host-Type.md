# 专有宿主机规格

专有宿主机规格标识其对应的计算、内存、存储、网络能力以及支持调度在该宿主机上的云主机实例类型。

以下为当前京东云在售的专有宿主机规格信息，不同地域可售卖规格不完全相同，请以实例创建页面所显示为准。具体在售实例规格类型根据不同应用场景可以分为：

* [通用型](instance-type-family#user-content-1)
* [内存型](instance-type-family#user-content-2)
