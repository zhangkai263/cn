# 备份管理

## 查看备份&手动备份

只有手动备份可以删除，自动备份不允许删除，进行中的备份不能删除。

备份如果失败或错误，鼠标移上去可显示错误原因。

根据用户在初始化备份计划中备份策略设置的备份类型显示物理备份或者逻辑备份，如果开启了增量备份，则会显示增量备份列表内容。

对应物理备份或者逻辑备份tab页，用户可以手动创建对应物理备份或者逻辑备份。
