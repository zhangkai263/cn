# 结束迁移任务

迁移类型为“结构迁移”、“全量迁移”时，数据迁移完成后，任务将自动结束，迁移类型为“增量迁移”时，需要手动结束迁移任务。

## 注意事项

- 迁移类型为“增量迁移”时，DTS服务将持续获取源库数据变更后复制到目标库，需要您关注增量迁移的进度。建议在结束确认前停止源库写入，确认目的库数据已追上源库后再结束迁移任务。
- 任务结束后将不能再次启动，请务必确认目标库数据库已追上源库。

## 操作步骤

1、进入任务列表页。

2、选择迁移任务，点击**结束。**
