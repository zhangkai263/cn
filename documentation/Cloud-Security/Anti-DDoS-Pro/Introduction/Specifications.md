# 产品规格

IP高防支持运营商三线（电信、联通和移动）和BGP线路的实例规格。

## 实例规格列表

| 规格列表 | 运营商三线防护能力 | BGP线路防护能力 |  
| :------ | :---------: | :---------: |
| IP数量    | 3个 | 1 到 100个 |
| 保底防护  | 5G 到 600G       | 10G 到 600G  |
| 弹性防护  | 0G 到 1100G      | 0G 到 1100G  |  	
| 业务带宽  | 100M 到 2000M    | 100M 到 2000M  |  
| 防护端口  | 60 到 150个      | 60 到 20000个  |  
| 防护域名  | 60 到 150个	     | 60 到 20000个  | 

最大防护能力达1.5T。
